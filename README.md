# Password Locker
Password Locker allows users to store their login credentials for the various platforms that they have. A user can display/delete/add credentials. 

## Prerequisites
- Have Git installed.
- Have a text editor installed e.g VS Code, Atom
### Technologies Used
- Python
- VS Code.
### Setup Installation
To run the application:-
1. Clone the repository to a folder in your machine using `https://github.com/joanwanjiku/passwordLocker`
2. Cd to that folder.
3. `code .`, if you are using VS code.
4. Run on your terminal
